# Installation

## Dependencies

Kaptive requires the following software and libraries, if installing
from source they will need to be installed and available in your path:

- [Python](https://python.org/) \>=3.9
- [Biopython](https://biopython.org/) \>=1.83
- [minimap2](https://lh3.github.io/minimap2/)

## Download and install Kaptive

From PyPI:

<!-- termynal -->
```bash
$ pip install kaptive
---> 100%
Installed!
```

From Bioconda:

<!-- termynal -->
```bash
$ conda install -c bioconda kaptive
---> 100%
Installed!
```

From source:

<!-- termynal -->
```bash
$ pip install git+https://github.com/klebgenomics/Kaptive.git
---> 100%
Installed!
```
